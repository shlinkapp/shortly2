## Design Context

### Users
Shortly is a public SaaS-style product for external users who need to shorten, share, and manage links quickly with minimal friction. The product also includes richer signed-in workflows for dashboard management, temporary email, API keys, and admin controls, so the interface should help users move from one-off utility to trustworthy account-based usage without feeling overwhelming.

### Brand Personality
Calm, professional, trustworthy. Beyond that baseline, the interface should also feel high-control and low-pressure: users should feel they can move quickly, understand consequences, and complete tasks without friction or visual noise. The tone should feel efficient and credible rather than playful, flashy, or overdesigned.

### Aesthetic Direction
Light mode first, with a restrained palette and clean visual hierarchy. Current signals in the codebase already point toward a neutral shadcn-based system with Geist typography, subtle borders, low-saturation surfaces, and simple motion. Keep that direction. Explicit anti-references: avoid generic AI SaaS template styling, avoid hacker/cyber-neon aesthetics, avoid overly heavy enterprise-admin visual weight, and avoid toy-like social-app playfulness.

### Design Principles
1. Prioritize clarity over novelty — primary actions, status, and link data should read instantly.
2. Build trust through restraint — use neutral color, disciplined spacing, and predictable patterns instead of visual hype.
3. Create a sense of efficient control — users should feel they can act quickly without losing situational awareness.
4. Keep flows lightweight — creation, copying, review, and admin tasks should feel fast, direct, and low-pressure.
5. Design for operational readability — tables, badges, dialogs, and empty states should support scanning and decision-making.
6. Use motion sparingly — prefer subtle transitions and always avoid effects that add noise or discomfort.
7. Avoid obvious template aesthetics — no glossy AI clichés, no cyber-security theatrics, no bloated enterprise heaviness, and no playful toy-app tone.
